# OG Images for Social Sharing

This folder contains Open Graph images for social media sharing previews.

## Required Images

Create the following images at **1200Ã—630 pixels** (recommended for optimal display):

| File | Page | Suggested Content |
|------|------|-------------------|
| `default.png` | Fallback | Cloud Native Latvia logo, tagline, Riga skyline |
| `home.png` | Home | Hero image with logo, "Breaking Through Cognitive Load" |
| `events.png` | Events | Calendar/event imagery with Kubernetes icons |
| `speakers.png` | Speakers | Microphone/speaker imagery with tech icons |
| `team.png` | Team | Team photos or community imagery |
| `sponsors.png` | Sponsors | Partnership/sponsorship themed |
| `swag.png` | Swag | Stickers and merchandise showcase |

## Design Guidelines

- **Brand Colors**: Rose (#d4567c), Burgundy (#8b1538), Pink Light (#fdf2f4)
- **Font**: Noto Sans (Bold/Black for headlines)
- **Include**: Cloud Native Latvia logo
- **Text**: Keep text large and readable (previews are often small)
- **Safe Zone**: Keep important content away from edges (60px margin)

## Tools for Creating OG Images

- [Figma](https://figma.com) - Free design tool
- [Canva](https://canva.com) - Easy template-based design
- [OG Image Generator](https://og-image.vercel.app) - Code-based generation
